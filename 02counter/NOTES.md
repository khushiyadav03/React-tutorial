# 02 Counter - Short Overview

## Kya implement kiya
- Counter app with increase, decrease, reset buttons.
- Min/max limit handle kiya.
- Counter ke hisaab se progress bar aur status message update kiya.

## Kya sikha
- `useState` se component state manage hoti hai.
- State change hone par React component re-render karta hai.
- Functional state update `setCounter(prev => ...)` latest value ke saath safe hota hai.
- Events like `onClick` user actions handle karte hain.
- Conditional rendering se UI state ke hisaab se text change hota hai.
