# Custom React - Short Overview

## Kya implement kiya
- Ek simple custom renderer banaya.
- React element jaisa object create karke real DOM element me convert kiya.
- Props ko loop karke DOM attributes ke form me set kiya.

## Kya sikha
- React internally UI ko object-like structure se describe karta hai.
- Renderer ka kaam element object ko browser DOM me convert karna hota hai.
- `type`, `props`, aur `children` React element ke basic parts hote hain.
