# 08 Context API Toggle Project - Short Overview

## Kya implement kiya
- Login form aur Profile component banaya.
- Login data shared context me save kiya.
- Profile component ne same context se user data read kiya.

## Kya sikha
- Context API props drilling avoid karne ke liye use hoti hai.
- `createContext` shared data box banata hai.
- Provider data ko child components tak available karta hai.
- `useContext` kisi component me context value read/update karne deta hai.
- Local form state aur shared app state alag concepts hain.
