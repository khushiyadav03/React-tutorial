# 07 React Router Project - Short Overview

## Kya implement kiya
- Multi-page React app with Home, About, Contact, User, Github pages.
- Header/Footer common layout me rakhe.
- Dynamic route `/user/:userId` banaya.
- Github page ke liye route loader use kiya.

## Kya sikha
- React Router SPA me page navigation handle karta hai.
- `createBrowserRouter` aur `RouterProvider` routing setup karte hain.
- Parent layout ke andar child pages `Outlet` me render hote hain.
- `Link` page reload ke bina navigation karta hai.
- `NavLink` active route ka style handle kar sakta hai.
- `useParams` dynamic URL value read karta hai.
- Loader page render hone se pehle data fetch kar sakta hai.
