# 09 Theme Switcher - Short Overview

## Kya implement kiya
- Light/Dark theme switcher banaya.
- Theme state context ke through share ki.
- Tailwind `dark:` classes ke saath UI theme change ki.

## Kya sikha
- Context me state ke saath functions bhi pass kar sakte hain.
- Custom hook `useTheme` context use karna easy banata hai.
- `useEffect` se `html` tag par `light`/`dark` class update ki.
- Controlled checkbox current theme ke hisaab se checked hota hai.
- Tailwind dark mode class-based theme switching support karta hai.
