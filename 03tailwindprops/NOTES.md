# 03 Tailwind Props - Short Overview

## Kya implement kiya
- Reusable `Card` component banaya.
- Same component ko different data ke saath render kiya.
- Tailwind CSS classes se card UI style ki.

## Kya sikha
- Props parent component se child component ko data bhejte hain.
- Props destructuring se `props.username` ki jagah direct `username` use kar sakte hain.
- Default/fallback value `btnText || "Learn more"` se set kar sakte hain.
- Component reuse React ka important pattern hai.
