# 06 Currency Project - Short Overview

## Kya implement kiya
- Currency converter app.
- API se exchange rates fetch kiye.
- From/To currency dropdowns banaye.
- Swap currency feature add kiya.

## Kya sikha
- Custom hook `useCurrenyInfo` API fetching logic ko reusable banata hai.
- `useEffect` currency change par naya data fetch karta hai.
- `fetch(...).then(...)` async API data handle karta hai.
- Controlled input parent state se connected hota hai.
- `useId` label aur input ko unique id se connect karta hai.
- List render karte time `key` prop important hoti hai.
