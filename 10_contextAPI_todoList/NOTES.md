# 10 Context API Todo List - Short Overview

## Kya implement kiya
- Full todo app using Context API.
- Add, edit, delete, complete todo features banaye.
- Todos ko localStorage me save kiya.
- Todo count, pending count, completed count show kiya.

## Kya sikha
- Context API actions aur state ko multiple components me share kar sakti hai.
- `useState` array state ko add/map/filter se update kar sakte hain.
- `map` se specific todo update hota hai.
- `filter` se todo delete hota hai.
- `useEffect` first load par localStorage read kar sakta hai.
- `useEffect` state change par localStorage write kar sakta hai.
- Custom context hook `useToDoContext` code ko clean rakhta hai.
