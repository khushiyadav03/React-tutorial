# 01 Vite React - Short Overview

## Kya implement kiya
- Basic Vite + React app setup.
- `App` component ke andar `Chai` component use kiya.
- Multiple JSX elements ko Fragment `<>...</>` me return kiya.

## Kya sikha
- React app `main.jsx` se start hoti hai.
- `createRoot(...).render(...)` app ko HTML ke `root` div me mount karta hai.
- Component ek normal JS function hota hai jo JSX return karta hai.
- Component ko export/import karke reuse kar sakte hain.
