# 05 Password Generator - Short Overview

## Kya implement kiya
- Password generator with length, numbers, and symbols options.
- Range input se password length control ki.
- Checkbox se options toggle kiye.
- Copy password to clipboard feature add kiya.

## Kya sikha
- Multiple `useState` values form options ko track kar sakti hain.
- `useCallback` function ko memoize karta hai.
- `useEffect` dependency change hone par password regenerate karta hai.
- `useRef` DOM input ko directly access karne ke liye useful hai.
- Controlled inputs me value React state se aati hai.
